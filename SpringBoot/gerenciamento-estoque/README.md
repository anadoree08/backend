# Sistema de Gerenciamento de Estoque SENAI

## Descrição
Sistema desenvolvido para controle de materiais e ativos patrimoniais da unidade escolar SENAI-SP.

## Tecnologias
- Java
- Spring Boot
- JPA
- Thymeleaf
- H2 Database

## Requisitos Funcionais

RF01 – Cadastro de materiais  
RF02 – Cadastro de categorias  
RF03 – Registro de movimentações de estoque  
RF04 – Controle de ativos patrimoniais  
RF05 – Login de funcionários por NIF  

## Requisitos Não Funcionais

RNF01 – Interface responsiva  
RNF02 – Banco de dados relacional  
RNF03 – Sistema web acessível via navegador  
RNF04 – Segurança de acesso via login
